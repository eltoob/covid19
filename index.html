How does it work?

  • You can click on any cell to mark it as infected
  • This model uses the classic SIR (suceptible, infected and recovered) method.
  • Every node is assumed to have some probabilty to get sick when surrounded by another sick node.
  • After an incubation period, a node is considered as recovered or removed and not infectious anymore.
